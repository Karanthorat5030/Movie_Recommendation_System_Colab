# Movie_Recommendation_System-_Colab
Movie Recommendation System 🎬  A Content-Based Movie Recommendation System built using Python and Machine Learning. It recommends similar movies using the IMDb Top 1000 dataset by applying TF-IDF Vectorization and Cosine Similarity on genres, descriptions, and cast information. Built with Python, Pandas, NumPy, Scikit-learn, and Google Colab.
